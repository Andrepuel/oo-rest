# Minimalistic Typescript Project Starter

### Features
- TSLint
- Basic Tests Structure (Mocha and Chai)
- Dev friendly

### Usage

#### Installing
```bash
$ git clone git@github.com:brunoventura/typescript-project-starter.git your-awesome-project
$ cd your-awesome-project
$ npm i
```

#### Using
```bash
$ npm run watch
```
